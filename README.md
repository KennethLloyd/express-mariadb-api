# express-mariadb-api

Simple NodeJS Express App with Sequelize as ORM

Requirements:
1. MariaDB 10+
2. Node 12+

To setup in a new local environment:
1. Make sure you have the correct Node version
```sh
$ nvm use
```

2. Install dependencies
```sh
$ npm install
```

3. Run APIdoc
```sh
$ npm run docs
```

4. Start the dev server
```sh
$ npm run dev
```
